## Comparision Operators

    >=   Greater Than or Equal to
    <=   Less Than or Equal to
    ==   isEqualTo
    !=   Not Equal To
    >    Greater Than
    <    Less Than

## if else elif

    if condition :
      do something
    else :
      do this

    // Example
    water_level = 10
    if water_level > 80 :
      print("Drain Water")
    else :
      print("Continue Fill")
